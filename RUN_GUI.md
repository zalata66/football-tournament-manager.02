# How To Run The GUI

1. Open `FootballTournamentManager.html`.
2. Read the Home page, then click `Open Tournament Setup`.
3. Add the tournament name, format, team count, point values, default match duration, teams, players, captain, and goalkeeper.
4. Optionally assign player tiers and positions, or use `Randomize` / `Level Randomize` to create teams from a player list.
5. After randomizing teams, optionally choose each randomized team's captain and goalkeeper in `Randomized Roles`.
6. Click `Start Tournament`.
7. Use the match timer if needed, then enter each match result and player statistics.
8. The final screen shows the champion and awards.

The HTML file is standalone. It runs in Chrome, Edge, Firefox, Safari, or any modern browser.

Use `Export Data` to download the tournament data as JSON, and `Import Data` to load it again later.

Use the `Dark Mode` / `Light Mode` button in the top bar to switch themes.

On phones, use the `Menu` button in the top bar to open navigation.

Formats included: Round Robin, Double Round Robin, Knockout, Double Elimination, and Two-Legged Knockout.
